### Features
* Written in C++20
* Using OpenGL 4.6 DSA
* Texture-loading via stb_image
* Model-loading via Assimp
* UI via ImGui
* Cubemap skybox
* Gamma correction and tone-mapping
* PBR (IBL is in progress)
* Post-processing via a kernel
* Using MSAA for anti-aliasing
